Anatomia

/* Primary CTA */
.e-btn-primary {
  background: var(--e-purple-600);
  border-radius: var(--e-r-pill);     /* pílula */
  box-shadow: var(--e-sh-cta);        /* glow + hairline */
  padding: 14px 28px;
  transition: all 300ms var(--e-ease);
}
.e-btn-primary:hover {
  background: var(--e-purple-700);
  transform: translateY(-1px);
}

Hierarquia

  • Primary: uma única ação por dobra. Roxo-600 sólido + glow.
  • Ghost: ações secundárias. Borda hairline, transparente.
  • Link: ações terciárias inline. Underline animado.
  • On-cream: exclusivo para dobras com inversão. Roxo-700 sólido (sem glow, fundo cream já contrasta).
