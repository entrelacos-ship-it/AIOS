01Fundamentos

Densidade emocional vence saturação visual.

Todo o sistema é construído sobre três decisões nucleares. Quem violar qualquer uma delas está fazendo outra marca, não Entrelaços.

As três decisões nucleares

Todo o sistema visual da Entrelaços nasce de três decisões. Não são preferências estéticas — são traduções diretas da tese estratégica para o plano visual.

A
Autoridade premium sem luxo vazio

Roxo denso domina. O laranja só aparece como acento — nunca preenche, nunca grita. A profundidade clínica é a estrela.

B
Contraste tipográfico imediato

Sans tight + italic-serif na mesma headline. É a assinatura visual: nunca substituir italic por bold, nunca usar serif em body.

C
Inversão cream entre dobras

Maior parte é dark premium. Manifesto e momentos-âncora rompem para cream — criando respiração emocional intencional.

Princípios não-negociáveis

01

Cada dobra cumpre função narrativa

Não há seção decorativa. Hero abre tensão, Manifesto rompe ritmo, Programas entrega caminho, Provas validam, Comunidade gera pertencimento.

02

Mono em labels, italic em destaques

JetBrains Mono em CAPS para eyebrows. Instrument Serif italic para palavras-âncora. Inter Tight para body. Nunca trocar.

03

Eyebrow sempre presente

Toda seção começa com eyebrow curto e mono — nunca direto na headline. É o ritmo que sustenta a leitura.

04

Glow de cor é a depth principal

Sombras coloridas (roxo translúcido) fazem o que sombras pretas fariam num design genérico. Inset hairline branco + glow = assinatura.

Geometria diferenciada por família

A forma comunica a função. Pílula = ação. 20px = card. Quadrado = entrada de dado. Não misturar — ver 07 · Raio & geometria.

Como detectar uma página fora da Bíblia. Se você bate o olho e vê: (a) gradiente roxo→azul ou roxo→rosa, (b) fundo branco puro, (c) mais de 3 elementos laranja, (d) input com cantos arredondados, (e) headline 100% sans sem italic-serif, (f) seção sem eyebrow → não é Entrelaços. É um SaaS roxo qualquer.
